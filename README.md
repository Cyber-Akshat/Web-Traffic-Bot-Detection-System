# Bot-Detector
This is the bot detector that protects from different attacks such as web scraping, API abuse and creating fake accounts.

# What is a Bot-Detector Expected to do?
- Scripted Form Submission
- Automated Login Attempts
- Scrapers
- Crendential Stuffing Attack
- Spam bots
- Headless Browser Automation
- API Abuse

# Score Risks, Not just detect! (Under DEVELOPEMENT)
- Score Cards (Demonstration Purpose only!)

| Score         | Meaning       | Action           |
| ------------- | ------------- | ----------       |
| 0 - 30        | Human         | Allow            |
| 40-60         | Suspicious    | Challenge        |
| 70-100        | Bot           | Very Likely a Bot|

# Protecting Areas (What you can use this tool for!)
- Login Areas
- Signup Page
- Contact Forms
- Checkout Pages
- API
- Admin Dashboards
